# Portfolio Website

## Description
This is a portfolio website designed to showcase projects, skills, and experiences. It serves as a personal branding tool and provides potential clients or employers with insight into the work done by the individual.

## How to Use
- Navigate through the website using the navigation menu at the top. 
- Explore different sections such as Projects, About, and Contact. 
- Each project will have a brief description and links to the respective project repository or live demo.

## Features
- Responsive design for mobile and desktop views.
- Project gallery featuring images and links.
- Contact form for inquiries.
- Easy navigation with a clean and user-friendly interface.

## Adding/Updating the Logo Image
1. Locate the logo image file in the repository, typically found in the `assets` folder. 
2. To change the logo, replace the existing image with a new one with the same filename or update the filename in the HTML file if using a new image. 
3. Ensure the new image is optimized for web (recommended size is under 500 KB).
4. Commit your changes and push to the repository to see the updates live on the website.
